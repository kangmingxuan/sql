# 算术运算符 #

这里一些有关 SQL 算术运算符如何使用的简单示例：

	SQL> select 10+ 20;
	+--------+
	| 10+ 20 |
	+--------+
	|     30 |
	+--------+
	1 row in set (0.00 sec)
	
	SQL> select 10 * 20;
	+---------+
	| 10 * 20 |
	+---------+
	|     200 |
	+---------+
	1 row in set (0.00 sec)
	
	SQL> select 10 / 5;
	+--------+
	| 10 / 5 |
	+--------+
	| 2.0000 |
	+--------+
	1 row in set (0.03 sec)
	
	SQL> select 12 %  5;
	+---------+
	| 12 %  5 |
	+---------+
	|       2 |
	+---------+
	1 row in set (0.00 sec)
